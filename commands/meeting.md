# AI 3자 회의 (Claude + GPT + Gemini)

사용자가 제시한 주제에 대해 Claude, GPT, Gemini 3개 AI가 **서로 반박하고 설득하는 실시간 토론**을 진행한다.

## 사전 조건

- Codex CLI 설치 및 `codex login` 완료
- Gemini CLI 설치 및 브라우저 로그인 완료

## 핵심 규칙

1. **의견 나열이 아니라 진짜 토론**을 한다. AI들이 서로 반박하고 설득해야 한다.
2. **모든 과정을 사용자에게 실시간으로 보여준다.** 각 AI의 발언을 받으면 즉시 출력한다.
3. **최소 3라운드** 토론 후 합의 또는 최종 정리를 한다.
4. 사업 연결 없이 순수 토론에 집중한다.

## 진행 순서

### Step 1. 주제 파악 및 토론 시작 선언

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎙️ AI 3자 토론 시작
주제: {사용자 주제}
참석: Claude · GPT · Gemini
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### Step 2. 라운드 1 — 초기 입장

**Claude가 먼저 입장을 밝힌다** (300자 이내). 출력 형식:

```
🔵 [Claude] 라운드 1
(의견)
```

**GPT 의견을 수집한다.** Bash로 codex exec 호출. 프롬프트:

```
당신은 AI 토론에 참석한 전문가입니다. 한국어로 500자 이내로 답변하세요.
[주제]: {주제}
[Claude 발언]: {Claude 의견}
Claude의 의견에 대해 동의하는 부분은 인정하되, 반드시 하나 이상 반박하거나 다른 시각을 제시하세요. 상대를 설득하려는 태도로 답변하세요.
```

받은 응답을 즉시 출력:
```
🟢 [GPT] 라운드 1
(응답)
```

**Gemini 의견을 수집한다.** Bash로 echo | gemini 호출. 프롬프트:

```
당신은 AI 토론에 참석한 전문가입니다. 한국어로 500자 이내로 답변하세요.
[주제]: {주제}
[Claude 발언]: {Claude 의견}
[GPT 발언]: {GPT 의견}
두 AI의 의견을 듣고, 동의하는 부분은 인정하되 반드시 하나 이상 반박하거나 새로운 시각을 제시하세요. 상대를 설득하려는 태도로 답변하세요.
```

받은 응답을 즉시 출력:
```
🔴 [Gemini] 라운드 1
(응답)
```

### Step 3. 라운드 2 — 반박과 설득

**Claude가 GPT와 Gemini의 의견에 반론한다** (300자 이내). 구체적으로 누구의 어떤 주장에 반박하는지 명시. 출력:

```
🔵 [Claude] 라운드 2 — 반론
(GPT의 ~주장에 대해: ...)
(Gemini의 ~주장에 대해: ...)
```

**GPT에게 라운드 2 반론을 요청한다.** 프롬프트에 이전 라운드 전체 대화 내용을 포함:

```
AI 토론 라운드 2입니다. 한국어로 500자 이내로 답변하세요.
[주제]: {주제}
[라운드 1 전체 대화]:
- Claude: {라운드1 의견}
- GPT(당신): {라운드1 의견}
- Gemini: {라운드1 의견}
[라운드 2 Claude 반론]: {Claude 반론}

Claude의 반론에 대해 재반박하세요. 동의할 부분은 솔직히 인정하고, 여전히 다른 부분은 구체적 근거로 설득하세요.
```

출력:
```
🟢 [GPT] 라운드 2 — 반론
(응답)
```

**Gemini에게 라운드 2 반론을 요청한다.** 동일하게 전체 대화 포함. 출력:

```
🔴 [Gemini] 라운드 2 — 반론
(응답)
```

### Step 4. 라운드 3 — 수렴 시도

**Claude가 합의점을 찾으면서도 양보할 수 없는 핵심을 정리한다.** 출력:

```
🔵 [Claude] 라운드 3 — 수렴
(인정하는 부분: ...)
(양보할 수 없는 부분: ...)
```

**GPT에게 라운드 3 수렴을 요청한다.** 프롬프트:

```
AI 토론 최종 라운드입니다. 한국어로 500자 이내로 답변하세요.
[전체 토론 히스토리]: {라운드1~2 전체}
[Claude 수렴 발언]: {Claude 수렴}

토론을 마무리합니다. 다른 AI들의 의견 중 수용할 부분은 솔직히 인정하세요. 하지만 여전히 양보할 수 없는 핵심 주장이 있다면 최종 근거와 함께 제시하세요.
```

출력:
```
🟢 [GPT] 라운드 3 — 수렴
(응답)
```

**Gemini에게 라운드 3 수렴을 요청한다.** 출력:

```
🔴 [Gemini] 라운드 3 — 수렴
(응답)
```

### Step 5. 최종 판정

Claude가 전체 토론을 분석하여 최종 정리한다:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 토론 최종 정리
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🤝 합의에 도달한 부분
- ...

⚔️ 끝까지 평행선인 부분
- Claude: ...
- GPT: ...
- Gemini: ...

💡 토론을 통해 진화한 생각
- (라운드 1과 비교하여 각 AI가 입장을 수정한 부분)

🏆 가장 설득력 있었던 주장
- (Claude가 판단한 이 토론의 MVP 주장과 이유)
```

### Step 6. 후속 토론
사용자가 추가 질문이나 새 주제를 제시하면 Step 1부터 반복한다.
사용자가 "더 해봐", "계속" 등을 말하면 라운드를 추가로 진행한다.

## 주의사항
- GPT/Gemini 호출 시 에러가 나면 해당 AI는 "응답 불가"로 표시하고 나머지로 진행한다
- 각 AI의 응답을 받으면 **즉시 사용자에게 출력**한다 (기다렸다가 한번에 보여주지 않는다)
- 토론이 너무 합의적이면 Claude가 일부러 도발적 반론을 던져 긴장감을 유지한다
