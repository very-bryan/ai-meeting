# AI 3자 회의 (Claude + GPT + Gemini)

사용자가 제시한 주제에 대해 Claude, GPT, Gemini 3개 AI가 **서로 반박하고 설득하는 실시간 토론**을 진행한다.

## 사전 조건

- Codex CLI 설치 및 `codex login` 완료
- Gemini CLI 설치 및 브라우저 로그인 완료

## 핵심 규칙

1. **의견 나열이 아니라 진짜 토론**을 한다. AI들이 서로 반박하고 설득해야 한다.
2. **모든 과정을 사용자에게 실시간으로 보여준다.** 각 AI의 발언을 받으면 즉시 출력한다.
3. **최소 3라운드** 토론 후 합의 또는 최종 정리를 한다.
4. **매 라운드 종료 후 사용자에게 참여 기회를 준다.** 사용자가 의견을 넣으면 다음 라운드에 반영하고, "계속" 또는 엔터만 치면 바로 다음 라운드로 진행한다.

## 진행 순서

### Step 1. 주제 파악 및 토론 시작 선언

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎙️ AI 3자 토론 시작
주제: {사용자 주제}
참석: Claude · GPT · Gemini · 사용자(옵저버/참여자)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### Step 2. 라운드 1 — 초기 입장

**Claude가 먼저 입장을 밝힌다** (300자 이내). 출력 형식:

```
🔵 [Claude] 라운드 1
(의견)
```

**GPT 의견을 수집한다.** Bash로 codex exec 호출. 프롬프트:

```
당신은 AI 토론에 참석한 전문가입니다. 한국어로 500자 이내로 답변하세요.
[주제]: {주제}
[Claude 발언]: {Claude 의견}
Claude의 의견에 대해 동의하는 부분은 인정하되, 반드시 하나 이상 반박하거나 다른 시각을 제시하세요. 상대를 설득하려는 태도로 답변하세요.
```

받은 응답을 즉시 출력:
```
🟢 [GPT] 라운드 1
(응답)
```

**Gemini 의견을 수집한다.** Bash로 echo | gemini 호출. 프롬프트:

```
당신은 AI 토론에 참석한 전문가입니다. 한국어로 500자 이내로 답변하세요.
[주제]: {주제}
[Claude 발언]: {Claude 의견}
[GPT 발언]: {GPT 의견}
두 AI의 의견을 듣고, 동의하는 부분은 인정하되 반드시 하나 이상 반박하거나 새로운 시각을 제시하세요. 상대를 설득하려는 태도로 답변하세요.
```

받은 응답을 즉시 출력:
```
🔴 [Gemini] 라운드 1
(응답)
```

**라운드 1 종료 후 사용자에게 참여 기회를 준다:**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💬 라운드 1 완료. 의견을 추가하시겠습니까?
   → 의견을 입력하면 라운드 2에 반영됩니다
   → "계속" 또는 엔터만 누르면 바로 라운드 2로 진행합니다
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

그리고 **여기서 멈추고 사용자 입력을 기다린다.**

- 사용자가 의견을 입력하면: 그 의견을 [사용자 발언]으로 추가하여 라운드 2에서 3개 AI가 모두 사용자 의견에도 반응하도록 한다.
- 사용자가 "계속", "넘어가", "다음", "ㄱ" 등을 입력하면: 사용자 의견 없이 바로 라운드 2를 진행한다.

### Step 3. 라운드 2 — 반박과 설득

**사용자 의견이 있는 경우**, Claude는 GPT/Gemini뿐 아니라 사용자 의견에도 반응한다.

**Claude가 반론한다** (300자 이내). 출력:

```
🔵 [Claude] 라운드 2 — 반론
(사용자의 ~주장에 대해: ...) ← 사용자 의견이 있을 때만
(GPT의 ~주장에 대해: ...)
(Gemini의 ~주장에 대해: ...)
```

**GPT에게 라운드 2 반론을 요청한다.** 프롬프트에 이전 라운드 전체 대화 내용 + 사용자 의견을 포함:

```
AI 토론 라운드 2입니다. 한국어로 500자 이내로 답변하세요.
[주제]: {주제}
[라운드 1 전체 대화]:
- Claude: {라운드1 의견}
- GPT(당신): {라운드1 의견}
- Gemini: {라운드1 의견}
- 사용자: {사용자 의견} ← 있을 때만 포함
[라운드 2 Claude 반론]: {Claude 반론}

Claude의 반론에 대해 재반박하세요. 사용자 의견이 있다면 그것에도 반응하세요. 동의할 부분은 솔직히 인정하고, 여전히 다른 부분은 구체적 근거로 설득하세요.
```

출력:
```
🟢 [GPT] 라운드 2 — 반론
(응답)
```

**Gemini에게 라운드 2 반론을 요청한다.** 동일하게 전체 대화 + 사용자 의견 포함. 출력:

```
🔴 [Gemini] 라운드 2 — 반론
(응답)
```

**라운드 2 종료 후 다시 사용자에게 참여 기회를 준다:**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💬 라운드 2 완료. 의견을 추가하시겠습니까?
   → 의견을 입력하면 최종 라운드에 반영됩니다
   → "계속" 또는 엔터만 누르면 바로 최종 라운드로 진행합니다
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**여기서 멈추고 사용자 입력을 기다린다.**

### Step 4. 라운드 3 — 수렴 시도

**사용자 의견이 있는 경우**, 수렴 시 사용자 의견도 고려한다.

**Claude가 합의점을 찾으면서도 양보할 수 없는 핵심을 정리한다.** 출력:

```
🔵 [Claude] 라운드 3 — 수렴
(인정하는 부분: ...)
(양보할 수 없는 부분: ...)
```

**GPT에게 라운드 3 수렴을 요청한다.** 프롬프트:

```
AI 토론 최종 라운드입니다. 한국어로 500자 이내로 답변하세요.
[전체 토론 히스토리]: {라운드1~2 전체 + 사용자 의견들}
[Claude 수렴 발언]: {Claude 수렴}

토론을 마무리합니다. 다른 AI들과 사용자의 의견 중 수용할 부분은 솔직히 인정하세요. 하지만 여전히 양보할 수 없는 핵심 주장이 있다면 최종 근거와 함께 제시하세요.
```

출력:
```
🟢 [GPT] 라운드 3 — 수렴
(응답)
```

**Gemini에게 라운드 3 수렴을 요청한다.** 출력:

```
🔴 [Gemini] 라운드 3 — 수렴
(응답)
```

### Step 5. 최종 판정

Claude가 전체 토론을 분석하여 최종 정리한다:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 토론 최종 정리
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🤝 합의에 도달한 부분
- ...

⚔️ 끝까지 평행선인 부분
- Claude: ...
- GPT: ...
- Gemini: ...

💡 토론을 통해 진화한 생각
- (라운드 1과 비교하여 각 AI가 입장을 수정한 부분)

🏆 가장 설득력 있었던 주장
- (Claude가 판단한 이 토론의 MVP 주장과 이유)
```

### Step 6. 후속 토론
사용자가 추가 질문이나 새 주제를 제시하면 Step 1부터 반복한다.
사용자가 "더 해봐", "계속" 등을 말하면 라운드를 추가로 진행한다.

## 주의사항
- GPT/Gemini 호출 시 에러가 나면 해당 AI는 "응답 불가"로 표시하고 나머지로 진행한다
- 각 AI의 응답을 받으면 **즉시 사용자에게 출력**한다 (기다렸다가 한번에 보여주지 않는다)
- 토론이 너무 합의적이면 Claude가 일부러 도발적 반론을 던져 긴장감을 유지한다
- **매 라운드 종료 후 반드시 멈추고 사용자 입력을 기다린다** — 자동으로 다음 라운드를 시작하지 않는다
- 사용자 의견이 들어오면 GPT/Gemini에게 전달하는 프롬프트에 [사용자 발언]을 반드시 포함한다
